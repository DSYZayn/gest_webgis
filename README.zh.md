# 如何使用示例项目
```sh
$ npm install
$ npm run dev
```
我推荐使用 [Visual Studio Code](https://code.visualstudio.com/) 作为编辑器，`node@^20.0.0 `作为运行环境，`npm` 作为包管理器。

> `pnpm`可能造成意料之外的问题，如果你遇到了任何问题，请使用`npm`代替或者随时提出issue。
